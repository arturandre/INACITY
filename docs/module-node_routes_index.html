(inner) getPanoramaForFeature(feature) → {Promise}

Receives a GeoJson Feature (not a FeatureCollection) and for each of its coordinates tries to find a panorama. This function returns the input feature with a new property(geoImages) with the same structure as the coordinates property and each value represents a StreetViewPanoramaData for each coordinate. e.g.: features[0].properties.geoImages[x][y][z] corresponds to the coordinate at features[0].coordinates[x][y][z]
Parameters:

feature Feature GeoJson Feature object with one or more coordinates
