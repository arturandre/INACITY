Class: Layer

UserSection~Layer(id, active)

Each layer is named after a MapMiner and a Feature concatenated by an underscore. Each layer contains a collection of regions named 'regions' and a list of features called featureCollection with all the features collapsed into a single list. Layer keeps track of vector features (e.g. Points, Lines, Polygons, ...) related with some Map Miner (e.g OSM) and Geographic Feature Type (e.g. Street)
