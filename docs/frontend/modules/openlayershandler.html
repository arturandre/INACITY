OpenLayersHandler¶

class OpenLayersHandler(HTMLDIVtarget, defaultTileProvider)¶

Responsible for handling several OpenLayers properties

Arguments:
  • HTMLDIVtarget (string) – The DOMElement div’s id that will be the container of map tiles, the map itself in other words
  • defaultTileProvider (OpenLayersHandler.TileProviders) – The tile provider as registered at TileProviders class member. for OpenStreetMap’s tiles, ‘google_roadmap_tiles’ for Google Maps tiles.
OpenLayersHandler.TileProviders¶

Collection of registered tile providers from OpenLayers

Examples:

//OpenStreetMaps tile provider
OpenLayersHandler.TileProviders.OSM

//Google road maps tile provider
OpenLayersHandler.TileProviders.GOOGLE_ROADMAP_TILES

//Google satelite with road maps tile provider
OpenLayersHandler.TileProviders.GOOGLE_HYBRID_TILES

See also

  • [OpenLayers Sources]{@link https://github.com/openlayers/openlayers/tree/v5.0.3/src/ol/source}
OpenLayersHandler.changeMapProvider(tileProvider)¶

Set the map tiles provider for displaying

Arguments:
  • tileProvider (OpenLayersHandler.TileProviders) – The tile provider as registered at TileProviders class member.
