GeoImageManager¶

class GeoImageManager(DOMImageId)¶

Responsible for displaying GeoImages from features.

Arguments:
  • DOMImageId (string) – The id of the image element (from DOM) that will be used to display the collected GeoImages.
GeoImageManager._findNextValidImage(startingIndex)¶

Tries to find the next valid image starting from “startingIndex”. If some valid GeoImage is found the [_currentIndex]{@link module:GeoImageManager~_currentIndex} is set to the index of the GeoJson found.

Arguments:
  • startingIndex (int) – Start the search from this index (possibly zero) until the end of the GeoImage’s collection
Returns:

GeoImage|null – If no valid GeoImage is found then returns null.
GeoImageManager._isValidJsonObject(testString)¶
Returns:Object|False – If the ‘testString’ is a valid json object then it’s returned otherwise “false” is returned.
GeoImageManager.autoPlayGeoImages(this)¶

Changes automatically the currently presented geoImage

Arguments:
  • this._autoPlayState (int) – Controls the state of GeoImageManager’s autoplay 0 - Stopped -> Will restart the GeoImageManager counter when started. 1 - Playing -> Can be stopped (reseted) or paused. 2 - Paused -> Will continue from the last presented GeoImage when restarted.
GeoImageManager.displayFeatures(fromStart)¶

Start the displaying of the current GeoImages collection

Arguments:
  • fromStart (boolean) – If true then the counter will be reset
GeoImageManager.geoimagescollectionchange¶

type: Array.<GeoImage>

Triggered when GeoImagesCollection (the set of geo images) changes

GeoImageManager.imagechange¶

type: GeoImage

Triggered when MultiLineString Features are joined together

GeoImageManager.invalidcollection¶

Triggered while trying to display a GeoImage from a collection without any valid GeoImage.

GeoImageManager.setCurrentGeoImagesCollection(newFeatureCollection)¶

Change the current GeoImage’s Collection being presented

Arguments:
  • newFeatureCollection (FeatureCollection) – A feature collection object with its features containing the geoImages as a property
Returns:

boolean – True if the change is successful, false if otherwise
GeoImageManager.validImages¶

Getter for [_validImages]{@link module:GeoImageManager~_validImages}; the number of valid images in the [_currentGeoImagesCollection]{@link module:GeoImageManager~_currentGeoImagesCollection}
