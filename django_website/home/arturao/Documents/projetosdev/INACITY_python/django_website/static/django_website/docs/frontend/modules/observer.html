Observer¶

class Subject(eventNames)¶

This class keeps a list of _observers and dispatch (notify) them about some event with some parameters

Arguments:
  • eventNames (Array.<string>) – All the event names this class dispatches
Subject.on(eventName, listener, opt_this)¶

Listens for an event named “eventName”

Arguments:
  • eventName (string) – The event name
  • listener (function) – The listener function
  • opt_this (object) – The object to use as this in listener
Subject.registerEventNames(eventNames)¶

Register at class level the eventNames that the subclass can dispatch events for and creates, also at class level, a dictionary of observers.

Arguments:
  • eventNames (Array.<string>) – The names of the events the subclass will dispatch.
